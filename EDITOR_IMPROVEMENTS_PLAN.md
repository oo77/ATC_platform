# План реализации улучшений редактора

## 1. Панель настроек для множественного выбора

### Что показывать при выборе нескольких элементов:

#### Заголовок

- Бейдж: "Выбрано элементов: N"
- Кнопки: Разблокировать все / Удалить все

#### Позиция и размер

- **Общие значения** (если одинаковые у всех)
- **Смешанные значения** (если разные) - показать "—" или "Смешано"
- Возможность изменить для всех сразу

#### Выравнивание

- Относительно холста (работает для всех выбранных)
- Относительно друг друга:
  - Выровнять левые края
  - Выровнять центры по горизонтали
  - Выровнять правые края
  - Выровнять верхние края
  - Выровнять центры по вертикали
  - Выровнять нижние края
  - Распределить по горизонтали
  - Распределить по вертикали

#### Общие свойства (только для текстовых элементов)

- Шрифт (если все текстовые)
- Размер шрифта
- Цвет текста
- Выравнивание текста

#### Действия

- Сгруппировать элементы
- Дублировать все
- На передний план
- На задний план

## 2. Кастомные размеры макета

### Интерфейс:

1. Выбор макета: A4 / Letter / Кастомный
2. Ориентация: Альбомная / Книжная
3. При выборе "Кастомный":
   - Чекбокс "Свои размеры"
   - Поля ввода: Ширина (px) и Высота (px)
   - Валидация: минимум 200px, максимум 5000px

### Реализация:

- Добавить тип `custom` в `TemplateLayout`
- Хранить кастомные размеры в `templateData`
- Обновить `LAYOUT_DIMENSIONS` для поддержки кастомных размеров
